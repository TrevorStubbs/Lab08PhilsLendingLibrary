## Description 
This is a C# console application that will allow the user to use a lending library. The user can ViewAll, Add, Checkout, and Return books. 
The user will also be able to view their books in their book bag.

---

### Getting Started
Clone this repository to your local machine.

```
$ git clone https://github.com/TrevorStubbs/Lab08PhilsLendingLibrary.git
```

### To run the program from Visual Studio:
Select ```File``` -> ```Open``` -> ```Project/Solution```

Next navigate to the location you cloned the Repository.

Double click on the ```Lab08PhilsLendingLibrary``` directory.

Then select and open ```Lab08PhilsLendingLibrary.sln```
